LOCUS AI — GITHUB PAGES UPLOAD

1. Open your existing GitHub repository.
2. Upload every file and folder from this package to a folder named docs.
3. In GitHub, open Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select your main branch and the /docs folder, then click Save.
6. Wait a few minutes for GitHub to publish the site.

Important: upload the contents of this package, including the assets folder and .nojekyll file.
